# IFTTT Maker Channel SocketIO Server

HerokuでNodeJS(Express)をベースにした、IFTTT Maker ChannelとSocketIOが連携するサーバーです。

IFTTT Maker ChannelでのHTTP通信をSocketIOに置き換えます。

## Heroku Button

以下のHeroku Buttonを押してHerokuアプリを作成しましょう。
